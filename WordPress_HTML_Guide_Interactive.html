⚠️

很常看到，但新手先不要碰

這些標籤雖然常見，但在 WordPress 中容易與佈景主題衝突，或導致版面崩壞。

<div>

它是無意義的容器，主要是給工程師寫 CSS 排版用的。

為什麼不碰：容易不小心把整個版面切壞，且沒有視覺效果。

<font>

這是非常古老的標籤，用來改顏色或大小。

為什麼不碰：現代網頁標準已淘汰，且會被 WordPress 編輯器過濾掉。

<table>

用來製作複雜的表格。

為什麼不碰：語法極其複雜（tr, td, th），容易漏標籤導致全站跑版。請直接用編輯器的表格工具。

學到這裡就夠了

作為內容工作者，你不需要懂 CSS，不需要會寫 JavaScript。 只要善用這份教學裡的標籤，你的文章結構就會比 90% 的人更清楚、更好讀。
如果遇到想要「把字體變大一點點」或「圖片想要飛進來」這類需求， 請找工程師或尋求佈景主題設定，不要自己在文章內硬改喔！
